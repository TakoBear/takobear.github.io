才剛發布Raspberry Pi II, 微軟馬上就跟進宣布Windows 10支援他啦！

Feb 4, 2015

這幾天內工程師圈的朋友跟Tako大概最常聊到的話題就是樹梅派(Raspberry Pi)的第二代發表吧!!
雖然說大家很常在討論開發上的細節，不過現在還有另一個振奮的消息，
微軟在官方部落格宣布Windows 10也可以在上面執行喔!!!! 不過也是要特定身分才能優先使用，
如果你有參與微軟的物聯網開發者計畫(Windows Developer Program for IoT) ，在今年年底時就能搶先試用囉!!!
在先前的微軟發表會中，微軟已經發布許多項Windows 10的新特色，
除了個人電腦運算能力外，對行動平台的開發也是微軟投入的一大重點，
現在我們確定的是，Windows 10的項目包含了以下範疇：
  • PC (點選了解: TakoBear 搶鮮試用放送心得)
  • 平板
  • 智慧手機
  • XBox
  • 物聯網IoT (點選了解: 微軟進軍擴增實境的秘密武器HoloLens)

沒錯，最後一項就是最近最火熱的IoT物聯網啊!

(下圖為TakoBear Windows 10試用預覽圖)
微軟這次加入物聯網的戰局，也是來自於他們先前發現在Maker社群中已經充滿許多很好創意的產品
也因此它們將Maker社群做為實驗基地與微軟的物聯網開發計畫結合，
這次樹梅派的整合，一方面因為新一代的樹梅派電腦變得更厲害，也是微軟想嘗試大舉入侵物聯網平台的一步啊!!
當然啊，各位開發者們一定也猜道，在樹梅派上面的Windows 10版本一定跟現有平台的版本不是不一樣的
在功能上考慮到平台限制會有許多限縮，如果你對微軟的物聯網開發計畫有興趣，
現在也可以直接申請加入物聯網開發者計畫喔!!
