連起床用電腦都懶? Google 終於放出Chrome遠端遙控電腦iOS版啦!! 從此用電腦就只要躺床上滑手機控制就行!

Jan 13, 2015

相信很多人都有遇過這樣的問題：「我想我得開電腦來看一下現在的影片，啊好麻煩，這樣我還得從床上爬下來又爬回去....」
這樣的問題應該很多人都曾遇過吧，去年四月的時候Google曾經出一款Chrome Remote Desktop的App
他主打的概念就是當你利用Chrome與Android手機就可以遠端遙控你家裡的電腦，
而非一定要用Teamviewer這樣的解決方案，但是iOS終於到今天也有一樣的方案推出啦!!
目前這款App已經在iPhone與iPad版都能使用下載，使用者只要搭配Chrome上的Web App設定丸藥使用的連接碼就行
  • iOS版: Chrome Remote Desktop載點
  • Chrome Web Store版: Chrome Remote Desktop載點
只要下載行動裝置與你的Chrome Web App都載好就能開始使用來建立連線了
至於如何使用，大家可以參考以下的圖片示意喔!!
由於提供的是Chrome上的解決方案，大家並不用擔心說要給Mac或PC使用，兩邊都能夠支援的!!
