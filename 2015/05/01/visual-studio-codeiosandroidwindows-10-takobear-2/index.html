微軟力推Visual Studio Code，不論是iOS/Android都能直接轉移到Windows 10上！ （含TakoBear簡易試用心得）

May 1, 2015

這一兩年來開發者很忙碌，忙著專注行動裝置的開發上，你不只要會寫 Objectiv C，還得學習新的語言 Swift，Android 也棄用 Eclipse 要學習 Android Studio，在這麼忙碌的情況下，有多少人還記得要在 Windows 上開發應用程式累積用戶呢？

寫一套 Code 就在所有平台應用這種幾乎天方夜譚式的神話，在今天以後可就要成真啦！！！

[cjtoolbox name=’google_ad’]

點選前往 Visual Studio Code

在這之前跨平台服務這種概念幾乎不會是從跟微軟有關的詞聯想得到，但從微軟這陣子積極的往做雲端服務發展腳步來看，一切倒也不是那麼意外了。

Tako 先來說說這個 Visual Studio Code 有多厲害，也跟過去以往的微軟做法有多不同吧！

雖然掛上了 Visual Studio 的名字，但說是完整的 IDE，其實它更像是Sublime Text這樣的一個文字編輯器

簡易的 Code 助手包含提示字元支援了 Android SDK、Objective-C、Web語法以外，旁邊也有包含像是 Call Stack、變數的整理等，看起來很像是 IDE 了？但是它比完整的 IDE 輕量化許多！

（Tako: 這樣應該讓大家比較能接受去下載它吧XD 因為是編輯器的關係，整套檔案下載也只要 62.7MB！！）

擁有近代開發程式人員必學的工具 Git（不知道Git流程的請看這）

在這個編輯器內同時就能使用Git指令操作程式碼環境，讓你在一個環境內就同時完成基本所需要的操作！

Tako 稍微試用了一下，這個是去年 TakoBear 推出的《Tickr》專案，以 iOS 來說比 Xcode 的開啟時間短（因為只是編輯器），在程式碼的階層、Syntax Highlight 上做得還不錯

左邊四個 Tab 依序內容是：

  • 檔案內容
  • 搜尋程式碼
  • Git 指令操作
  • Debug 用

Debug 用的部分，也就包含程式碼可以執行（指的像是網頁開發應用）、可以追加斷點（Breakpoint），查看變數數值情況、追蹤 Call Stack，介面相當簡潔看起來也相當舒適，很難想像這樣的使用者體驗會是從微軟所推出的軟體開發服務XDD

說了這麼多，那Visual Studio Code這樣的工具，微軟還對他寄與什麼樣的重責大任呢？

  • 開發者可以直接用你的網站開發工具（Ex：Ruby on Rails），把你的網站打造成一個 Web 用程式
  • 程式可以直接移植到 Windows Store

如果你原本是 Android 的開發者，你可以用原本的 Java、C++ 放到 Windows 10 的手機上。在 Windows 10 中，將能夠直接包含到一個 Android 子系統，專門用於執行這樣的 Android程式（你可以想成，只要寫了 Android 版本，就可以直接發到 Windows 10 上了！）

做了 Android，竟然連 iOS 也能吃得下！透過 Visual Studio Code，你原本用 Objective C 寫的 App 只要經過部分微調也能發佈到 Windows 10 上了！！

微軟這次的做法真的相當酷，對於開發人員來講等於幾乎不用負擔額外的學習、管理成本就能額外獲得一個平台的使用者量，這簡直是天大的福音！不過關於這部分目前 Tako 手邊還沒有 Windows Phone 可作測試，之後有機會會幫大家發表試用心得的！
