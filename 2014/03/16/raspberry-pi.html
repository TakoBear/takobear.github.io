Raspberry Pi安裝後設定

Mar 16, 2014

大家好！ 上次介紹了如何安裝印象檔（以Raspbian為主） 接下來我們就要介紹有關Raspbian的設定～有興趣的版友千萬別錯過喔!
如果上次是使用官方推薦的超容易安裝方式 那重新開機後就會出現上述畫面 一些讓Raspberry Pi更加容易使用的設定如果不是用上述方法，而是更為硬派的做法（把img檔寫入SD卡中） 那在出現上述畫面之前 先介紹Unix-like底下最為方便的多人遠端連線SSH 如果不知道這是什麼工具的話，這邊不說高深的介紹 簡單的說，就是加密版的telnet（誤 （telnet最常見連線的伺服器就是telnet://ptt.cc） 如此一來，就是要用SSH連線裝好的Raspberry Pi伺服器 什麼，啥時就架好一個SSH Server? 放心，Raspbian預設就將SSH裝好並提供服務了 小編先推薦幾個windows底下的SSH連線工具 pietty putty Xshell putty是老牌的SSH/Telnet工具，而pietty則是根據putty原始碼再開發的 Xshell是一套功能強大的商業軟體，但是學術或家用則是免費使用 如果是第一次使用SSH工具，小邊推薦pietty
下載完後啟動 ...IP...那啥？可以吃嗎？ 要怎麼知道IP呢？ 小編這邊先介紹DHCP的方法 如果Raspberry Pi是網路接在一台IP分享器、無線AP之類的後面 與Raspberry Pi相同IP分享器後方的windows電腦上網不需要使用PPPoE的話 請打開流覽器輸入192.168.1.1或192.168.0.1 進入後可能需要輸入帳號密碼，如果不知道請檢查IP分享器或無線AP上，應該有預設帳密
因為各家的使用介面都不同 重點放在區域網路（LAN）這部份，類似的文字也沒關係，稍微找一下吧 如果幸運的話，應該可以看到raspberrypi跟對應的IP如果找不到或是家中上網方式是使用PPPoE的話 小編這邊很抱歉請各位拿出螢幕（HDMI）、鍵盤（USB）接上去吧
輸入IP，port是22，按下連線吧
確認ssh-rsa金鑰，放心按是（輸入yes） 提示輸入預設的帳號（login as）：pi 密碼（password）：raspberry
很抱歉小編這邊使用的SSH工具是Xshell，不過請放心，步驟都是一樣的 來吧，輸入第一個指令 $ sudo raspi-config
總算可以開始設定了 上述可以看到九個選項，分別是 1.Expand Filesystem 將SD所有空間都拿來給OS使用（反正這張SD卡就跟著pi吧，全部拿去用吧） 2.Change User Password 更改使用者pi的密碼（為了安全跟好記憶，小編建議一定要將預設改掉） 3.Enable Boot to Desktop 開機後是否進入桌面 （這部份看各位的需求決定吧，如果不要的話可以再開機後手動輸入startx） 4.Internationalisation Options 設定使用語系、地區等等設定（這部份小邊稍候再談） 5.Enable Camera 啟動相機模組（小編沒買相機模組，跳過） 6.Add to Rastrack 加入Rastrack大家族（這部份看各位的需求決定吧） 7.Overclock 超頻（如果散熱問題處理得不錯，可以試試看） 8.Advanced Options 進階設定（這部份小邊稍候再談） 9.About raspi-config 關於 raspi-config（可以看看哦）
Internationalisation Options底下還有三個子設定 I1.Change Locale 更改使用語系 I2.Change Timezone 更改時區 I3.Change Keyboard Layout 更改鍵盤配置
使用語系可以選擇多個en_US.UTF-8、zh_TW、zh_TW.UTF-8等 不過最後主系統語系建議用en_US.UTF-8
時區當然是亞洲、台北
http://en.wikipedia.org/wiki/IBM_PC_keyboard 鍵盤配置小編是參考上述網頁，選擇美式104鍵（這部份看各位的需求決定吧）
美式鍵盤當然是選擇美式英文（Englsih US）
http://en.wikipedia.org/wiki/AltGr_key AltGr key? 似乎是歐系鍵盤才有的 選擇No AltGr key
http://en.wikipedia.org/wiki/Compose_key Compose key? 還是歐系鍵盤才有的 當然選擇No Compose key
詢問是否要設定Ctrl+Alt+Backspace為強制關閉X server（GUI桌面）？ 這部份看各位的需求了
Advanced Options底下也有一些設定 A1 Overscan 掃描使用的螢幕的細節並配合之 A2 Hostname 更改這台Raspberry pi的主機名 A3 Ｍemory Split 記憶體分配 因為Raspberry pi是CPU跟GPU共享記憶體，所以要好好分配記憶體 如果常用桌面或觀看影片，那GPU就給大一點 A4 SSH 啟用或關閉SSH服務 A5 Update 更新主機完成後，Raspberry Pi會提示重開機，重開後就完成上述所有的設定 可以開始拿Raspberry Pi做事了! 想知道後續更多有關介紹 還請繼續關注TakoBear! 或是來信到我們的信箱: takobearx@gmail.com
