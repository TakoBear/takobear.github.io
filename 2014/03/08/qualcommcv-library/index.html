Qualcomm專門為行動裝置開發的CV Library

Mar 8, 2014

image from http://www.mwdd.com/
一說到Computer Vision的Library 我們一定會馬上聯想到 OpenCV 在之前我們也介紹過如何設定OpenCV 不論是在電腦上跑 或是 app的開發 但是OpenCV 裡面有的東西太多了 有時候反而會降低他的效能 有鑒於這一點 這邊我們要介紹另外一個Computer vision 的library FastCV
FastCV是由Qualcomm所推出了 專門為行動裝置的而產生的library FastCV理所當然的把常用到的功能都收入其中囉而且可以讓下列這些功能 加入到以相機為主的App喔 其中包含了 1. 手勢辨識 2. 臉部辨識 臉部追蹤 人臉偵測 3. 文字辨識以及追蹤 4. 擴增實境 (Augmented Reality) FastCV適用在所有已ARM架構下的處理器 但是針對Qualcomm自家開發的處理器 Qualcomm's Snapdragon processor (Snapdragon 800, 600, 400, 200) 有做特別的優化 所以會有更好的表現哦 下面是簡單的影片介紹
