[Opensource] Google OpenSource 新計畫! Android開發者也能播Youtube Live直播啦!

Oct 29, 2014

來自Google Youtube的WatchMe開源計畫項目今天發布了新的功能特色! Android App的開發者們, 之後如果要在Android App內加入有關播放Youtube直播的功能 也就是市面上例如HTC新推出的RE Camera或是Sony推出的Live功能的話, 現在有Google提供的功能可以直接使用啦!!!
Youtube WatchMe Github載點透過這項開源計畫, 第三方開發者們例如Takobear我們, 只要去上面Github載點把工具下載移植到要開發的專案內就能使用了, 目前這個Youtube WatchMe專案使用了幾個API:
  • YouTube Data API v3
  • YouTube Live Streaming API
  • Google Play Services
  • Plus API
基於使用的API特性應該能跟滿足大部分作直播需要的需求, 不過因為Google也提到, 這項計畫目前還在實驗階段, 直接使用的話還是會有一段陣痛期的喔!
