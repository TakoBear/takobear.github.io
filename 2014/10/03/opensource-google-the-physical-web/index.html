[Opensource] Google 宣布The Physical Web計劃成立, 讓你隨處都可用到智慧裝置!

Oct 3, 2014

想像一下在不遠的未來, 當你走到一台販賣機, 或是你要訂一片Pizza, 租一台車, 可以多方便? 如果是現在的生活, 大家通常都要下載比如說55688的特製App來叫車這種方式? 如果你有一百種店家的服務要使用那你可能要下載100個類似這樣的服務, 但你其實只會用到一兩個功能而已? Google 今天發表的The Physical Web project就是讓你可以使用任何一種智慧裝置而且你不需要另外去下載App就能操作喔!
點選前往Physical Web Github project這項計劃其實是相當有遠見的, 比如說最近Apple跟飛利浦合作推出的Hue, 大家就必須要另外下載一個Hue的App, 那假設說之後公車站也有App, 那你也必須要下載公車站對應的App, Google想做的就是讓每個人都可以跟各式各樣的裝置溝通, 但你不需要另外下載每個裝置所專屬的App!! 不過要注意, 這項計劃並不是說要取代以後物聯網所有的App, 如他目標所說的, 他只是要增加讓人們與這些產品互動的方式, 而不是每次要使用以前都先下載App的這樣方式
這樣的概念要怎麼做到呢? 其實只要你的智慧裝置能夠上網, 並且擺放上網頁的話就可以了, 這項計劃的概念有舉幾個比較實際的例子:
  • 公車站要顯示下一班公車抵達的時間
  • 停車場, 販賣機可以讓你快速腹脹
  • 任何一間店面都可以提供線上服務, 不用過多的裝置成本
  • 快速租車
以往這類服務大家在台灣應該也有裝過類似的App吧, 但如果只是要用某些功能, 那大家之後也許就能用這樣的方式連到網頁而不需要特別下載App就能使用了, 如果有需要更進一步的服務功能才需要下載App, 讓這項功能與App開發者們做更深度的連接, 物聯網的時代, 看來又要因為開源往前大躍進一步啦!
