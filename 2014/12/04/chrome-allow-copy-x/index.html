[Chrome]看網頁時不能使用右鍵? Allow Copy 破解擋滑鼠、複製文字、防止反白等功能!

Dec 4, 2014

這幾年雖然越來越少，但大家在逛網站的時候應該還是會遇到一種很討厭的網頁，
你其實也沒打算做什麼或是只是想把內文分享給朋友一段話，但是當你要選取的時候卻發現網站把右鍵鎖住
或是把整個網頁都不能讓你選取的討厭事情吧!!
今天Tako介紹的這款外掛Allow Copy就是讓大家可以不要被這些網站捆住啦!
點選前往Chrome Web Store 下載Allow Copy

點選安裝新增後，他會要求一些檔案的權限，基本上就是為了讓你操作網頁要用的
按下確認就行，安裝完成以後，旁邊會多出一個小按鈕，之後在逛網頁時需要複製右鍵就按他一下
點選以後就能對你目前的網頁解鎖成功了!!
當然，Tako還是強調，介紹這樣的工具並不是希望大家去成為一個盜文貼文的人
而只是希望大家能有更好的閱讀體驗，如果對於對方網站文章有意刊載在其他平台記得先向對方詢問同意喔!
