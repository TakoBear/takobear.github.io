[iOS 8] 從Objective-C到Swift : 你的第2支Swift程式 - 如何在Swift使用UILabel以及UIButton (製作按鈕與標籤)? 

Jul 5, 2014

上篇教學中Bear介紹了大家怎麼在Swift中做出UIView, 這次就來教大家怎麼使用UILabel以及UIButton啦!
其實跟上一篇的Swift創建UIView幾乎一樣 這裡比較特別的是UIColor自行調配顏色的方法是將參數寫在UIColor的括號內 文字對齊的 enum 也改成 NSTextAlignment 之後再用點呼叫不同屬性 這跟Swift的結構有關，所以Swift的enum基本上都改為這種形式 方便與否見仁見智，Bear是覺得新方法滿不錯的！
來看看呈現結果吧!!!!!
現在大家基本功應該都有了，是不是對Swift越來越熟悉了呢！根本與objective-c大同小異呀！ 接下我們要實作一個UIButton，讓大家更了解怎麼使用selector!與addView相同，addTarget也是用點來呼叫 而Swift最大的差異就是obj-c 的 @selector( function ) 改成字串就可以呼叫了 至於function需不需要多加冒號：就看function有沒有要帶入sender了 function內的prinln()跟obj-c 的NSLog 有相同的功能，可以印出字串，而要印出物件的內容的話Swift是使用 \(物件) 寫在字串內，取代了obj-c 中%@的用法
讓我們多按幾次來看看log的結果吧！
看到這邊有沒有更理解了Swift啊!? 別急, Bear之後還有更多Swift教學會放的XDDD
