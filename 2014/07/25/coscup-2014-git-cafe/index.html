COSCUP 2014 - 什麼是Git Cafe? 程式碼開源託管服務新崛起!

Jul 25, 2014

上個週末TakoBear來到全台灣的開源人年會大拜拜, 認識不少新朋友, 很高興能在同領域範圍內結識不同的好友
雖然回到上班工作後還是一樣忙碌, 儘管來得晚, Tako還是依約一一來寫有想要介紹分享的內容啦!!
第一篇就先介紹這個最近在開放程式碼內算是火紅的名字 -- "Git Cafe"
在開始介紹以前, 相信很多朋友還不知道Git Cafe是什麼吧, 以下先做簡單的介紹:
引述 GitCafe blog 的內容：
Git是目前世界上最流行最优秀的项目版本控制系统之一，
Cafe的意思为咖啡馆，象征着程序员文化。
在GitCafe这个平台上，开发者可以轻松的在线协作共同开发出一个又一个开源或者私有项目。
通过GitCafe，开发项目的控制与团队管理将变得方便与有效。

Gitcafe 創始人 Thomas Yao 在今年 coscup 2014 的議程演講『讓開源更好地促進教育發展』中提到：
「在傳統教育方面，多數學校教學課程的內容落後，資源匱乏，
而且教授課程的老師不重視新興技術。所以網路上的信息平等，傳播速度加快造就了線上教育。」

GitCafe 並不希望只是單單作為一個程式碼託管服務，而是能夠影響社會更多層面的平台。
像是之前就有結合企業舉辦過 Online Hackathon ，大公司不需要再花許多時間打廣告，
就能夠得到許多應用與回饋，而參與的開發者也有實際項目能夠應用所學。

此外 GitCafe 也已經與上海，香港的學校合作成立 Open Innovation Lab(oil)。
他們的特色是什麼?
其實GitCafe 大部分的功能還是圍繞在 git 上，
有使用過 GitHub 服務的人想必很快就會上手，這邊先不再多作說明，
基本上就是另一種Git Server的服務啦!! 作為一個開源服務, 接下來就要介紹GitCafe 的特色內容囉
私信功能：在個人頁面中可以直接發送私信互相交流，在個人的私信頁面可以查看回覆。
徵才廣告：在左下角可以看到中國科技公司的徵才廣告。
專案列表：
專案的分類主要分為你的公開專案、私人專案、關注的專案，
如果你 fork 其他人的專案，會是放在公開專案下，而且專案名稱下會顯示是從哪裡 fork 的。
社群分享：GitCafe 提供社群分享連結，方便開發者快速分享專案在社群網站上。

最後, 其他還有什麼功能呢?
Git Cafe有專門與 GitHub Pages 對應的 GitCafe Pages 服務。
目前GitCafe 免費用戶擁有 512MB 的空間使用，超過的話每 512MB 每天需要付 4GitCoin。
GitCafe 私有項目相關的收費方式與 GitHub 包月制不同，是以天數計算：點選了解詳費詳細規則。

如果想要從將 GitHub 上自己的公開專案移到 GitCafe，
你除了可以在 git 增加 remote 外，GitCafe 也提供了匯入 GitHub 專案的功能噢！

整體來說, Tako相當推薦這個服務, 不只是程式碼託管的服務可以讓開發上變得很有彈性
省去許多自己架設伺服器的困難與高成本,
對於很多新創團隊來說, 不像以往的開發模式是只透過把所有資訊封閉在自己的團體內
適度的open source給世界認識自己團隊的貢獻也是相當重要的,
Git cafe兼具這兩項優點真的是很不錯!!
