微軟...真的是猜不透你啊!! Windows Phone 8.1 更新, 加強與Apple Passbook整合!

Apr 16, 2014

最近Windows Phone 8.1更新了! 這次是一個比較大的更新變動,
本來Tako周遭很少在使用Windows Phone的朋友, 不過這次的更新中有一項相當有意思的功能
讓Tako真的是猜不透微軟的思路..只好讓大家一起看看了XD
點選前往Tom Warren的Twitter

這次更新的內容中, 由這位Tom Warren老兄做了相當多的介紹
其中幾點更新內容很有意思, 讓Tako帶大家稍微看看吧!

Windows Phone 8.1 supports Apple's Passbook natively. I've been using it for flights. Works perfectly pic.twitter.com/MFScqvEVU7

— Tom Warren (@tomwarren) April 14, 2014
第一個一定要提的就是這玩意...
這個App名為 "Microsoft Wallet", 功能上嘛, 就是你可以把有合作對象的航空公司機票, 優惠券或是展覽門票等
收錄在這項應用程式內, 沒錯就跟Apple的Passbook一模一樣, 不過根據這位老兄所說,
可不只是像而已, 而是Works perfectly!!! 直接有支援Apple的Passbook!

使用者可以用它來讀取原本iPhone上有儲存好的票券並下載下來,
用法完全跟Passbook一樣, 不過啊, 蘋果官方可沒有正式承認與微軟有合作,
不確定是不是微軟單方面為了增加這項程式的應用廣度才支援Passbook的,
這兩間對手公司又是合作又是競爭的, 真的很讓人猜不透XD

The new Windows Phone 8.1 lock screens look great. Not ready to use just yet though pic.twitter.com/YgBaSCCmLj

— Tom Warren (@tomwarren) April 14, 2014
螢幕鎖, 私心說覺得這畫面似乎比Apple的螢幕鎖來得有質感啊..XD

Windows Phone 8.1 screen share to PC pic.twitter.com/Y5IytIPIlC

— Tom Warren (@tomwarren) April 14, 2014
手機畫面Mirror到PC上, 雖然iOS, Android上都有支援這種功能或有類似的App,
不過實際應用上...Tako目前還真想不到實質上的應用啊..
倒是之前有看到有人在類似技術去玩Wii 之類的遊戲

The Windows Phone 8.1 podcasts app now works outside the US. Finally. pic.twitter.com/phtqPYjid5

— Tom Warren (@tomwarren) April 14, 2014
終於支援美國以外的地區可以Podcast!

Weird Windows Phone 8.1 bug. It's Tuesday today not Monday. See calendar tile pic.twitter.com/AfrRMv0beM

— Tom Warren (@tomwarren) April 16, 2014
不過似乎也有bug, 日期上有顯示錯誤, 看來還是有些地方要努力加強!!!

整體來說, Tako周遭雖然使用Windows Phone的人少, 不過像是Lumia系列的還是有收到許多不錯的評價
不喜歡蘋果也不喜歡Android的朋友可以考慮一下噢!!
