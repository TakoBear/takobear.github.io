[開箱文] Pixy camera

Apr 18, 2014

還記得之前介紹過的Pixy camera嗎 號稱可以同時偵測幾十個物件 以及擁有50fps超高處理速度的Pixy camera
在三月底的時候已經陸續出貨 Bear手邊已經有一台囉 接下來就要簡單介紹給大家看看是不是真的跟影片上說
那麼厲害囉
打該Pixy盒內 看到的只有模組跟一條線以及一些螺絲 線的話 是給Pixy連接Arduino使用 螺絲的話主要是拿來
與pan/tilt組裝時會用到 當然也可以拿來做一些簡單的固定的動作
正面的話 應該沒什麼好介紹的 不過還是有簡單幾點要提醒大家的
1. Camera 是可以轉動的 拿來對焦用的 就像是大家的單眼相機手動對焦的功能一樣 所以當你發現你的畫面怎麼很麼模糊的時候記得調整一下相對應的焦段
2. 其實Pixy的camera是可以換的, Pixy所使用的m12 camera 搭配上 1280*800的image sensor
在這個網站下 http://www.m12lenses.com/Board-Lenses-s/12.htm

背面的話有幾個地方需要注意一下的
1. 左上角的話有一個mini USB的孔 主要是拿來接電腦的時候用的 (Pixy內不會附mini USB - USB的cable 要自己準備)
2. 右上角的10pin的孔 是拿來接上裡面附的那條線 以及Arduino中間有個6pin的地方
3. 右下方有兩個地方要注意
3.1 首先是兩排各有3 pin的地方 是拿來接servo的 Pixy可以直接將detect到東西的位置 幫你做好轉換以及輸出到servo的動作 左邊的話是接pan的部分 也就是上下的仰角
3.2 接下來是白色的部分 主要就是讓你接電源 6v-10v

Bear也買了官方的Pan/tilt Mechanism 主要是因為Bear太懶 貪圖一點方便
組裝好以後 就會像這樣
接下來就是 軟體的部分 從官方下載PixyMon 對於Mac的使用者的話 馬上就可以用了
(Windows 的使用者的話 可以參考官方網站的使用步驟)
剛開始的時候Bear一直無法讀取到camera資料 如果你遇到跟Bear一樣的問題
除了換一個usb port以外 跟Bear一樣 換一條線 有很多人遇到跟Bear一樣線材的問題
接下來 就是如何使用PixyMon了
首先在狀態列上面的 action 選擇 Set signature 1
預設可以儲存七個不同的顏色 (如果Bear 記得沒錯的話 顏色是存在HSV color space)
選擇完以後 你可以框選指定的顏色在畫面中 比如說我們框選一個黃色的區域 然後我們拿另外一個黃色的物件 做測試
當我們想要清除儲存的顏色時 直接點選restore default parameters
結論:
Pixy的處理速度真的很快 抓物件的準確率也還算不錯 不過有很多false positive(也就是偵測到很多不是同樣顏色的物件)
此外Pixy的影像 白平衡影響滿大的 在室內使用黃光的時候 整個白平衡幾乎會跑掉 所以很多顏色看起來會變得更接近
使得整個畫面幾乎都會變成被偵測到的區域 (Bear之後會陸續補上一些其他的圖)
