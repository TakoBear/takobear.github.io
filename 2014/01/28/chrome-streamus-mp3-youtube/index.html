[Chrome] Streamus - 不用下載MP3了! Youtube直接變成音樂播放器!

Jan 28, 2014

相信大家也很多人會使用Spotify的服務, 不過有時候Spotify還是會有沒完全收錄到的歌曲
最後還是要回流來用Youtube搜尋...不過現在有人已經開發出了新工具! 讓你的Youtube變成音樂廣播電台
而且是內嵌在Chrome內, 用起來相當方便實用喔!!
這款Streamus目前還沒有在Chrome Web Store上架, 想要使用它的話
先點選這裡前往下載
進到連結後會看到如此簡潔的畫面, 點選中間的綠色按鈕 "Installed"後就能下載囉!
安裝時會看到以下畫面
新增完成以後, 會在你的chrome右上角多上一個小icon, 一個S的符號(參考本文首頁圖)
Streamus的功能相當多元, 充滿了很多小驚喜, Tako將會一個個介紹給你怎麼用,
讓聽歌這件事變得更輕鬆更符合你的習慣!

1. 使用Streamus找尋歌單
首先左上角的搜尋按鈕, 按下去以後, 可以輸入你想要找的Youtube影片,
例如Tako很喜歡陳奕迅Eason的歌(趁機宣傳), 打上Eason就能看到以下畫面
點選歌曲的時候會出現三個icon, 分別是播放, 加入到Stream列表, 丟棄選項
按下+號以後右邊會直接跳出歌曲, 就被加入清單了噢!
清單很多的時候長按可以調整播放順序(參考下圖)
2. 直接在Chrome 內搜尋歌曲
只要依照這方法輸入:
streamus + [空白鍵] + 你要搜尋的歌曲名稱
就能直接尋找清單了, 是不是很簡單方便呢?

3. 智慧廣播電台Radio
智慧電台的功能只要打開底下的Radio後,
當目前的歌播放完畢,
Streamus就能自動幫你到Youtube找尋類似相關歌曲並直接加到清單內, 跟DJ一樣幫你不斷播放你喜歡的歌曲
另外不能少的也有包含可以幫你儲存目前的播放清單
是不是相當簡單好操作呢?
