有人反應 Bear 寫了太多 iOS 教學文…

在醞釀多日+工作需要的情況下，Bear 決定有別於 App 的開發教學內容

接下來開始一系列的 Android 開發板與 Android 開放原始碼分析心得!!!

如果你不知道 Android 電視盒是什麼，可以參考以下連結: * Google影音殺手 : Chromecast * 快播小方智慧電視盒簡單動手玩

要進行Android的開發, 首先你要先挑選一個適合的開發板

不過不像是 Raspberry Pi 或是 Arduino 已經在業界學界知名

Android 開發板可是相當混亂!!!

光是規格比較表就已經相當玲琅滿目了…….根據對岸的資料顯示，如果你需要 Android 開發板的性能比較，會有相當多的資源可以比較

有需要的話 Bear 再提供資源給大家下載!!

這邊 Bear 能給的建議是: 1. 依據你要開發的用途 2. 公司產品的定位方向, 3. 已經確認好要使用的核心

等來作為挑選的基礎

不過大家如果嫌這樣太麻煩，又想要簡單地做出一個 Android 電視盒的基本功能的話

這邊 Bear 強烈推薦這款 WaxBerry 還有 Tiny 4412 了 !!

如果你想做 Android 電視盒的研究  - > WaxBerry 是你最佳選擇

如果你希望做 Android 平板研究     - > Tiny 4412 配上螢幕也是一個不錯的選擇

首先在露天拍賣上就能買得到了!

他的圖樣大致如下:

前往露天拍賣購買

這塊板子的核心是 RK3066，Rockchip 出的

與前面連結所說的 RK3188 是同間公司出的核心喔!

在說優點前

先說說這板子有什麼缺點:

  1. 官方論壇文件有點混亂 (前往官方網站論壇)
  2. 簡體字，不過這東西還是內地做得好，畢竟人多資料也多.....
  3. 讀卡機目前無法偵測到

文件混亂會導致你在編譯(Build)程式碼的時候時常遇到問題

這邊記得大家拿到板子後，要跟原廠 (台灣只是代理進貨)拿到最新的 Source code 後進行編譯才行

另外購買時取得的光碟中的說明文件不全，基本上沒什麼參考價值……

V2 的版本較佳，詳細編譯與下載檔案方式 Bear 會另外再開一篇詳細介紹

那缺點說完了, 來說說優點吧!!

  1. 支援 HDMI，也就是你能用電腦螢幕輸出
  2. 預設支援6隻USB，連 USB Dongle 也能偵測到
  3. 可以偵測到行動硬碟, SATA 等
  4. 支援有線網路，PPPoE，Tethering (也就是說你喜歡的話讓它當成一台路由器 Router 使用也行...)
  5. Code base 中修改的部分也相當于 Sample Code 可以直接參考修改!

當然價格也是相當實惠，只需要 2600，比一檯 Android 手機相對來說便宜了太多!!!!

另外一台同樣值得推崇的, 是這檯 Tiny 4412

前往露天拍賣購買

這邊說說他的優缺點吧

先說缺點:

  1. 這邊從露天買到的板子配上光碟內的程式碼編譯後會有開機後不能進入休眠模式的問題，需要跟原廠取得檔案後重新編譯才能解決 (前往官方論壇討論區)
  2. 比較貴，價格是 WaxBerry 的兩倍
  3. 對硬碟支援度低，程式碼要再 Porting
  4. 沒辦法接有線網路，因為它就是一顆平板電腦........

優點的話如下:

  1. 官方文件相當齊全而且完整，基本上都是只要看圖說故事就行了
  2. 這個是 Samsung Note 2 的核心喔!! 想知道韓國人怎麼做得可以從這邊參考他們的做法......
  3. 直接配有 LCD 螢幕可操作

比較兩者一下

基本上他們都有在編譯時候會遇到的問題

Bear 之前在公司光是編譯他們還有聯繫廠商就花了快一個月的時間….

下篇教學時 Bear 會放上參考來源

但 Bear　個人相當推崇 WaxBerry，實在是因為他支援的東西太多了!!!!

再加上價格又很實惠, 他們在 vold 還有 netd 的修改實在是很不錯!

(不清楚 Bear 這邊在說什麼的，Bear 會再之後好好說明)
