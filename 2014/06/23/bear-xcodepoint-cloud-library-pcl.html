Jun 23, 2014

隨著kinect越來越多不同的應用之後, 2D的資料處理以及呈現已經略顯不足,
PCL 跟OpenCV很像是一個open source library 主要是處理 3D的 電腦視覺的問題
PCL 內包含了一些Feature estimation, surface reconstruction, registration, model fitting, segmentation的演算法

在這邊我們就不一一介紹每個模組了 這篇主要是要介紹要如何在Xcode下使用PCL 這個Library

在這邊我們使用pre-compiled binaries
首先先到 http://www.pointclouds.org/downloads/macosx.html 下載 PCL 以及其 Dependencies
