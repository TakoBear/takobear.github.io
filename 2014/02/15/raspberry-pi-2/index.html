替raspberry pi安裝作業系統

Feb 15, 2014

繼上次raspberry pi硬體介紹後 接下來要替raspberry pi安裝一個實用的作業系統 但是raspberry pi畢竟是台電腦 安裝方式、驅動程式...等等一大堆問題怎麼處理呢？ 所幸它的開發公司英國的樹莓派基金會已經替懶惰的小編提供一系列作業系統供小編選擇 不用像傳統的安裝過程要自己從頭慢慢弄 接下我們跳轉到它的官方網頁下載OS(作業系統)吧 http://www.raspberrypi.org/
上圖是利用google方式，畫面中第一個選項就是raspberry pi的官方網站 進入之後，馬上點選Downloads這個選項
目前raspberry pi提供更為簡易的安裝方式 就是上圖"New One"，不是，打錯了 是"New Out Of Box Software" 這方法很方便又快速 先下載NOOBS_v1.1.zip 解壓縮它，然後把解壓縮出來的檔案全部複製到SD卡中 將SD卡裝到raspberry pi上 開機！完成！
開機後會出現上圖畫面 可以選擇上述五種作業系統 小編這裡選擇Raspbian，一來這是raspberry pi基金會推薦 二來Raspbian是Debian系列，而Debian是Ubuntu的父親（誤） 而Ubuntu是小編常用的Linux
安裝完成了 三張圖就可以說明這安裝多麼容易，比Windows好裝多了（誤）
哎呀，小編錯了，看起來還是沒有比Windows容易多少 這部份的設定小編下篇再談
上述這方法雖然很容易很方便，但是有個致命缺點 raspberry pi必須先接上鍵盤（USB）、螢幕（具有HDMI） 一般來說，很少人擁有兩個以上的鍵盤、螢幕吧 或是懶得準備鍵盤、螢幕，那接下介紹比較古老的做法，只要有網路線（具有DHCP區網）就大丈夫回到官方網頁 http://www.raspberrypi.org/ Downloads頁面有個Raw Images 這些都是完整的OS，只要把img檔寫入到SD中就完成安裝了 這邊小編順便介紹raspberry pi目前支援的OS 目前最大宗都是Linux，有目前三大主流
基於Debian 上述兩者Raw Image，都是基於Debian的Raspbian 差異在程式編譯一個是用硬體浮點（快、相容性差），一個是用軟體（函式庫）浮點（慢，相容性高） 簡單的說，沒有編譯上特別需求就選硬體就好了 另外，小編後續示範還是用Raspbian
基於Arch Linux的Arch Linux ARM（廢話） 小編的前輩說過，一個真正的linuxer應該就是要用Arch Linux或是Gentoo 這表示，使用這兩個得具有深厚的使用經驗，有門檻的 不過這兩個的自由性可不是其他Linux distribution所比擬
基於RedHat（Fedora）的Pidora 是非常著名的商用Server選擇的OS至於RISC OS是ARM公司專為ARM CPU所寫的OS 它具有與硬體高度相容性、快，不像Linux那般臃腫 小編這裡式沒有特別嘗試，有興趣的可以自行測試看看 開始安裝前 用Windows先準備以下工具 Win32 Disk Imager http://sourceforge.net/projects/win32diskimager/ 這是在Windows底下將img檔寫入SD卡的工具 來，把剛剛下載的ZIP檔解壓縮，插入SD卡（將此格式化成FAT32） 點擊執行Win32DiskImager.exe
選擇要安裝的img檔與SD卡的磁碟機代號（別選錯了，會格式化資料） 點擊Write，泡杯咖啡，優哉得等它完成 （如果SD卡裡面有之前為了raspberry pi所安裝的OS，可以選擇Read備份或分享） 完成後，將SD卡放進pi中，插上網路線、電源開機
如果像小編一樣，手頭剛好有台Linux（這邊用Ubuntu為例） 想裝一下Geek的話... 第一件事情就是到官方網頁抓Raw Image 解壓縮它 $ unzip ~[放置的資料夾]/[檔名].zip 插上SD卡，掛載後檢查SD卡磁碟代號 $ df -h 看看哪一個最像是SD卡的容量，或是掛載在/media、/mnt底下 舉例是"/dev/mmcblk0" 或 "/dev/sdd"之類的 然後卸載它（如果是/dev/sdd1） $ umount /dev/sdd1 可能需要用到最高執行者權限（加個sudo試試看） 最後將img檔寫入SD卡中 $ sudo dd bs=4M if=~/[放置資料夾]/[檔名].img of=/dev/sdd 一樣泡杯咖啡，優哉得等它完成
如果厭惡key指令的話 可以安裝image writer $ sudo apt-get install usb-imagewriter 執行image writer $ sudo usb-imagewriter   or   $ sudo imagewriter 選擇img檔跟磁碟機代號，寫入 其實就是用GUI的方式執行 $ sudo dd bs=4M if=~/[放置資料夾]/[檔名].img of=/dev/sdd 完成後，將SD卡放進pi中，插上網路線、電源開機呼，總算弄好安裝了 至於開機後用SSH遠端raspberry pi來設定它，這部份小編下次再介紹! 期待看到多消息, 請密切關注TakoBear! 或是來信到信箱takobearx@gmail.com!
