Bear實驗室: 如何在台灣使用iTunes Radio? (含如何申請美國Apple ID)

Oct 22, 2013

iOS 7 中一個重點功能就是iTunes Radio
它可以讓你免費去串流iTunes Store所擁有的巨大資料庫
除了iPhone及iPad外，連iTunes 11.1也能使用喔!
目前這項功能限定在美國使用，不過台灣的朋友想搶先體驗的話也相當簡單喔
首先你需要更換商店到美國的App Store
要使用這項服務你必須先有一份美國的Apple ID
而且是不需要信用卡的

之後就是切換到美國App Store上囉
切換美國App Store後, 選一款免費的App
比如說點選Apple Store
點選 Free 選項後
會詢問你是否有Apple ID, 沒有的話就選擇Create Apple ID
之後會要求填一連串問題
經過這樣的步驟後, 最後會問填選信用卡資料
可以選擇"None"的資訊
地址的話, 可以參考這份資料: http://bbs.weiphone.com/read-htm-tid-424402.html

這邊我們重點是申請到美國Apple id後的操作
就先不多談了!

接下來就是先登入美國Apple ID
然後會看到以下畫面
選單上面就會出現廣播的選項!
接下來只要點下"廣播"的按鈕就會看到以下畫面..
點下聆聽廣播吧!
之後就能選擇你要的廣播電台,可以選擇任何歌手、歌曲、唱片，
iTunes Radio 就會開始替你選擇適合的歌曲播放。


你還可以選擇讓電台播放多一些較熱門的、不同曲風的，還是較冷門的歌曲。


iTunes Radio 目前有分免費和收費版，
免費版會有一些廣告穿插; 跳過歌曲的次數有限制，
但整體來說還是個相當讓Bear滿意的音樂串流服務!
