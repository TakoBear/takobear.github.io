Oct 5, 2013

大家晚安！！

Tako&Bear 的 App 好相報又來了！

今天要介紹的同樣也是 Mac 上使用的 App: SourceTree

注意！本篇適用於程式開發人員或有意使用版本管理者，一般用戶可跳過

這款App簡單來說就是一個gui的方式可以讓你管理 Git 有關的 repo 包含現在最有名的 GitHub 以及 Bitbucket

其實這邊 Bear 比較多都是用它在追蹤 log 以及檢查 merge、branch 的情況，畢竟有圖還是比較方便嘛！
