熱血! Github捐慈善培育年輕人寫程式

Oct 5, 2013

Tako跟Bear平日教學文都會放上的網站: github 這次大動作的表達出對教育的熱忱啦! Github代表參加慈善活動捐出巨額資金希望培育年輕一下寫程式的能力 希望能讓下一代能加快學習我們失敗的經驗從中進步! (Tako跟Bear都從上面獲益良多啊!)
"No one’s born knowing how to code." (沒有人生來就知道如何寫程式.) 但是某些情況下, 學習寫程式還是需要某程度的資金資源的: 取得電腦的機會, 買有關程式設計的書籍, 網路能力, 從前輩獲得指導等等 這些事情我們習以為常, 甚至視為理所當然的事情, 對某些孩子來說卻未必如此.Github希望能促進城市的資訊傳輸, 特別舉辦了一個“OctoGala”活動來贊助 Year Up 進行教育年輕人寫程式的活動 其實不用多說什麼 StackOverflow, Github這兩大網站已經大大解決我們工程師們平日所面對的問題資源 有事上Google, 沒事也查一下Google讓我們已經比過去成長的速度還要快了 但是如果大家整體知識都成長, 我們接下來進步的幅度也會越來越大 希望這類活動能喚起更多人來投入!
