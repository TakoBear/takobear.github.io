[Mac] Keynote Web view: 如何把網頁放到PPT內並維持即時更新狀態?

Dec 10, 2013

使用keynote 制作精美的投影片的時候，
是不是常常都會遇到想把網頁放進去，
但又希望我們想要看到的是最即時的網頁畫面，以前是不是都認為辦不到呢?

在keynote 8.0的版本的時候，
蘋果有內建一個功能是可以插入Web View 並且可以勾選是否要自動更新的強大功能!!!

但是這個功能在新版的keynote 9.0 卻消失不見了！！！
無法在插入的選項中看到...這點讓Tako感到相當難過.........

在Google 了許久之後發現
這項強大的功能在Keynote 9.0已經被移除了........


不過!!!!
此功能雖然沒有辦法在插入的地方看到
但依然被keynote所支援 在這邊我們就要教大家怎樣將這項功能復活
先下載 此Web view檔案

點選以下連結下載檔案:
web_view.key.zip
File Size: 597 kb
File Type: zip
Download File
解壓縮後 開啓裡面的檔案 可以看到蘋果最新的官方網站的畫面
此時我們點選該view之後
再點選右上角的檢閱器 -> 超連結
並將 URL 換成你想要放置的網站 並勾選自動更新
比如說我們將URL 改成 TakoBear的首頁 http://www.takobear.weebly.com/index.html
完成後 我們就可以看到 最新的首頁 而且每當網頁有更新時 Web View 便會自動更新
利用這個方法，就可以將在keynote 9.0版本消失的功能復活了!!!!!!

希望對於在keynote 9.0 版本裡面找尋Web view功能的人
以及想要將網頁嵌入在投影片裡面的人有所幫助
